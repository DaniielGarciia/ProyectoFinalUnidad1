# ProyectoFinalUnidad1

El desarrollo de esta actividad debe enfocarse prinicipamente en la interfaz del proyecto final. Este sera el primer prototipo del sitio web.


El sitio web a construir debe contener las siguientes características:

Página de Inicio:

Se debe crear un index.html (listo)

Este debe contener una barra de navegación con las opciones de redirección a la página Inicio y Página 2. (Listo)

Un carousel con mínimo 3 imágenes. (listo)

Debe contener tres secciones de contenido, que incluya contenido multimedia como imágenes, videos, párrafos, formularios, listas, etc. (Este contenido debe ser responsive a celular, tablet y escritorio por medio del sistema de columnas de Bootstrap).

Al final de la página se debe construir un footer, el cual se dividirá en 3 espacios para información del creador, Redes sociales (No reales) y otros links de interés.

La siguiente imagen es una simulación de cómo se debe visualizar esta página (descargar archivo: Visualización completa)

Página dos:
Se debe crear otro HTML.
Este debe contener la misma barra de navegación con las opciones de redirección a la página Inicio y Página 2(Nombre opcional).
Debe contener minimo tres secciones de contenido, que incluya contenido multimedia como imágenes, videos, párrafos, formularios, tablas, listas, etc. (Este contenido debe ser responsive a celular, tablet y escritorio por medio del sistema de columnas de Bootstrap).
Al final de la página se debe construir el mismo footer del inicio, el cual se dividirá en 3 espacios para información del creador, Redes sociales (No reales) y otros links de interés.
Al finalizar, tenga en cuenta que el sitio web debe contener hojas de estilos en CSS para la edición de colores, tamaños, fuentes, etc. para una correcta personalización del sitio web.
